Getting Started

Installation to start using design system Quantum

Quick Start

Mulai dengan memasukkan CSS dan JavaScript siap-pakai Quantum tanpa memerlukan langkah-langkah pembuatan apa pun.

  1. Sertakan file css Quantum Sertakan juga tag <meta name="viewport"> untuk perilaku responsif yang tepat di perangkat seluler 
                                                    <!doctype html>
<html lang="en">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">

    <title>Quantum Demo</title>

    <link href="http://quantum.sevima.com/assets/stylesheets/css/main.css" rel="stylesheet">
</head>
<body>
    ...
</body>
</html>
  2. Sertakan JS Quantum 
    Berikut seluruh JS yang bisa dipakai di versi ini:
    <!-- [START] core script -->
<script type="text/javascript" src="https://quantum.sevima.com/assets/jquery/jquery-3.6.1.min.js"></script> <!-- JQuery hanya untuk sementara ini -->
<script type="text/javascript" src="https://quantum.sevima.com/assets/js/core/app.js"></script>
<!-- [END] core script -->
